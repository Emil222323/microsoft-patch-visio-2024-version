- Expanded Content Library: Includes over 250,000 shapes and updated templates that adhere to industry standards like BPMN 2.0, UML 2.5, and IEEE.
- Modernized Design: Features a refreshed interface with a transparency effect and the new Office 2024 color scheme, including full Dark Mode support.
